Unicode(*) superscript 💩

Unicode has superscript versions of numerical characters. These are however not always rendered correctly. This is a test page, using your browser's default font.

These don't all line up the same way:

x⁰, x¹, x², x³, x⁴, x⁵, x⁶, x⁷, x⁸, x⁹

As a result, some of the following lines look like x^(ab), others look like x^a^b, and others just look broken.

x⁰⁰, x⁰¹, x⁰², x⁰³, x⁰⁴, x⁰⁵, x⁰⁶, x⁰⁷, x⁰⁸, x⁰⁹

x¹⁰, x¹¹, x¹², x¹³, x¹⁴, x¹⁵, x¹⁶, x¹⁷, x¹⁸, x¹⁹

x²⁰, x²¹, x²², x²³, x²⁴, x²⁵, x²⁶, x²⁷, x²⁸, x²⁹

x³⁰, x³¹, x³², x³³, x³⁴, x³⁵, x³⁶, x³⁷, x³⁸, x³⁹

x⁴⁰, x⁴¹, x⁴², x⁴³, x⁴⁴, x⁴⁵, x⁴⁶, x⁴⁷, x⁴⁸, x⁴⁹

x⁵⁰, x⁵¹, x⁵², x⁵³, x⁵⁴, x⁵⁵, x⁵⁶, x⁵⁷, x⁵⁸, x⁵⁹

x⁶⁰, x⁶¹, x⁶², x⁶³, x⁶⁴, x⁶⁵, x⁶⁶, x⁶⁷, x⁶⁸, x⁶⁹

x⁷⁰, x⁷¹, x⁷², x⁷³, x⁷⁴, x⁷⁵, x⁷⁶, x⁷⁷, x⁷⁸, x⁷⁹

x⁸⁰, x⁸¹, x⁸², x⁸³, x⁸⁴, x⁸⁵, x⁸⁶, x⁸⁷, x⁸⁸, x⁸⁹

x⁹⁰, x⁹¹, x⁹², x⁹³, x⁹⁴, x⁹⁵, x⁹⁶, x⁹⁷, x⁹⁸, x⁹⁹

Why do I care? Some Wikipedia pages use numerical superscript characters and can lead to mistakes!

* it's likely that this is a font or font-rendering issue affecting certain platforms. I can reproduce across different browsers on Mac OS X and iOS.
