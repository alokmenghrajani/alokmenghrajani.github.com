pour programmer chez soi

Si vous voulez programmer en C chez vous sur Windows, il vous faut simplement avoir un compilateur C et un éditeur de texte. Il y a plusieurs compilateur C disponibles:

  • Visual Studio de Microsoft (c'est payant, et c'est un compilateur different de gcc).
  • MinGW, c'est version minimaliste des outils GNU pour Windows. Vous pouvez télécharger MinGW gratuitement.
  • Installer Cygwin, vous aurez ainsi pleins d'outils Unix sur votre Windows et entre autre GCC. Cygwin est librement téléchargeable et d'installation facile.

    Note: Il y a des différences entre le gcc de cygwin et celui de mingw.

  • Acheter le CD de GNU Generation (Association d'étudiants de l'EPFL pour la promotion de GNU/Linux et des Logiciels Libres), pour 5 Chf. Vous pouvez aussi télécharger l'ISO du CD et le graver vous même.

Vous pouvez télécharger un éditeur de texte si vous n'en avez point:

  • Emacs pour Windows
  • UltraEdit

Vous pouvez aussi installer Linux. C'est gratuit et c'est mieux que Windows ;)
