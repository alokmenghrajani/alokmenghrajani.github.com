Javascript Music Box

A virtual music box plays a Twinkle-Twinkle inspired tune. As the hand crank turns, a brass drum rotates. Metal pins pluck tuned steel teeth.

The audio is rendered using Web Audio Spatialization (18 nodes, one per tooth). Move around to experience the sound change direction.

Model: hand made using Blender
Code: claude.ai (Opus 4.7)
