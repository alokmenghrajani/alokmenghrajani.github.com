Alok's camping checklist

Edits welcome via Github PRs!

Basics

  • Swiss knife.
  • Flashlight.
  • Sunglasses and/or hat.
  • Sunscreen + insect repellant.
  • Mesh clothing (to protect against insects).
  • First-aid kit.
  • Rain jacket + backup warm clothes.
  • Backpack + daybag.
  • Tent + sleeping bag + tarp + stakes.
  • Swimwear.
  • Reusable water bottle and/or camelbak.
  • Lighter + camp axe + gloves.
  • Toothbrush + toothpaste. Floss.
  • Nail clipper + tweezers.
  • Soap + Shampoo. Cotton swabs.
  • Laundry bag.
  • Cell phone + charger and/or power bank.
  • Permits/printouts (fishing, camp ground, directions, etc.).
  • Toilet paper + baby wipes.
  • Hand sanitizer.

Food related

  • Espresso maker + ground coffee beans.
  • Cooler.
  • Trash bags.
  • Plates + cups + silverware.
  • Stove + gas.
  • Pots + pans.
  • Salt + pepper + spices.
  • Paper towel roll.
  • Foil roll.
  • Dish soap + sponge + drying towels.
  • Oven mitt.

Extras

  • GoPro.
  • Board games + deck of cards.
  • Balls + frisbee.
  • Quarters for shower and games.
  • Binoculars.
  • Pleasure reading and/or Kindle.
